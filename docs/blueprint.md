# **App Name**: Nexus University

## Core Features:

- Secure Authentication & Profiles: User registration, login, and Google authentication powered by Supabase. Manage basic student, teacher, admin, and club profiles with assigned roles.
- Real-time Campus Navigator: An interactive map displaying the real-time locations of classrooms, labs, and other key campus facilities to assist new students.
- University Readiness Score: Calculates and displays a student's 'University Readiness Score' based on self-reported and validated academic and job market preparation activities.
- Personalized Recommendation Engine: Utilizes student interests and fields to recommend relevant courses and workshops. An AI tool will provide tailored suggestions for career readiness, like GitHub tricks or CV creation.
- Unified Notification Hub: Receive real-time notifications about important updates from university administration and various clubs.
- Admin & Club Management Panel: A basic dashboard for super admins to create teacher and administrative accounts, and for club managers to post announcements and notifications.

## Style Guidelines:

- Color scheme is light, evoking an open, inviting, and professional academic environment. Primary color: Confident, forward-thinking blue (#1F93CC) for navigation elements and key actions. Background color: A soft, subtle light blue-grey (#ECF3F6) for content areas, ensuring readability. Accent color: A vibrant yet harmonious blue-green (#66D6C0) for highlights, alerts, and calls to action.
- Headlines and prominent text will use 'Space Grotesk' (sans-serif) for a modern, slightly technical yet clean aesthetic. Body text will use 'Inter' (sans-serif) for excellent readability across different screen sizes and substantial content blocks.
- Clean, contemporary line-art icons that align with a minimalist and futuristic campus theme. Icons should be clear and immediately recognizable.
- A streamlined and grid-based layout prioritizing clarity and efficient information consumption. Ample whitespace to reduce cognitive load and enhance focus on key data points like scores and recommendations. Emphasis on responsive design for seamless experience across devices.
- Subtle, deliberate animations to guide user attention and provide feedback, such as smooth transitions for map interactions, notification arrivals, and dynamic score updates, contributing to a polished and futuristic feel.
